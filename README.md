# 🧹 Inventory Filter

> A lightweight Minecraft Forge mod that automatically removes unwanted items from your inventory.

![Minecraft](https://img.shields.io/badge/Minecraft-1.20.1-brightgreen)
![Forge](https://img.shields.io/badge/Forge-1.20.1-orange)
![License](https://img.shields.io/badge/License-MIT-blue)

## 📖 About

**Inventory Filter** lets you configure which items should be automatically deleted the moment they enter your inventory. Perfect for tech modpacks where you constantly pick up unwanted byproducts and junk.

---

## ✨ Features

- **9 filter slots** — configure up to 9 item types to filter out
- **Automatic deletion** — filtered items are removed instantly on pickup
- **Persistent config** — settings are saved to the item itself, take it anywhere
- **Multiplayer ready** — works on servers
- **Lightweight** — no performance impact

---

## 🔧 How to Use

1. Craft the **Inventory Filter** item
2. Right-click to open the filter GUI
3. Click a filter slot, then click any item to add it to the filter
4. Close the GUI — filtering is now active!

To remove an item from the filter, click its slot with an empty hand.

## ⚙️ Installation

1. Install [Minecraft Forge 1.20.1](https://files.minecraftforge.net/)
2. Download the latest `.jar` from [Releases](../../releases)
3. Place it in your `mods/` folder
4. Launch the game!

---

## 🛠️ Building from Source

```bash
git clone https://github.com/tmageko/inventory-filter-mod.git
cd inventory-filter-mod
gradlew build
```

The compiled `.jar` will be in `build/libs/`.
